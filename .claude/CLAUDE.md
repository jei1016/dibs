# Dibs Project Instructions

## Package Manager

This project uses **pnpm**, not npm. Always use `pnpm` commands:

- `pnpm install` (not npm install)
- `pnpm run check` (not npm run check)
- `pnpm run build` (not npm run build)
- `pnpm add <package>` (not npm install <package>)

## Monorepo Structure

This is a pnpm workspace monorepo. Packages are in the `packages/` directory.
